# Fake News Detector

A beginner-friendly web application that detects fake news
using simple rule-based text analysis.

## Features
- Sensational keyword detection
- Excessive punctuation analysis
- Capital letter usage check
- Confidence score generation

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
The application analyzes the given news text using predefined
rules and assigns a confidence score to determine whether the
news is likely fake or real.
